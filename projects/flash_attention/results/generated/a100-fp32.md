# A100 FP32 FlashAttention Canonical Benchmark

来源 CSV：`projects/flash_attention/results/raw/a100-fp32.csv`

协议：CUDA Event，warmup 10，iterations 50，repeats 3 取 latency 中位数；seed 1234。

环境：NVIDIA A100 80GB PCIe，SM 8.0，driver 610.43.02，commit `68bc4f9713a5b0cf902b3d3d302472a233fb2a72`。

> `Async Δ = 100 × (T_tiled / T_async - 1)`：正数表示 Async 更快，负数表示回退。它是 speedup delta，不是常规延迟增幅；ncu 时间不进入本表。
